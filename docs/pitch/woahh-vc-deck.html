Founding merchants → proof → flywheel.

  • Invite-gated launch. Sign-up requires a founding-access code (live in code) — scarcity + curation.
  • Founding offer: first ~20–25 merchants get a free subscription + zero commission, in writing. The foregone ~$300/mo each is bought back as testimonials, referrals, and the seed of the /eat marketplace.
